⚙️ À propos du projet

Ce projet implémente un pipeline complet d'apprentissage automatique (Machine Learning) visant à détecter des défauts mécaniques (défauts de translation) en analysant des séries temporelles issues de multiples capteurs.  Plutôt que d'appliquer un modèle sur des données brutes, l'architecture logicielle de ce projet se concentre sur l'Ingénierie des Caractéristiques (Feature Engineering) et la synchronisation de flux de données asynchrones.
